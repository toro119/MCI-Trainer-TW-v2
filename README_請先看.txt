MCI Trainer TW v2｜重新整理版

這個版本是全新專案，不需要覆蓋舊專案。

主要功能：
- 教官建立6位數房間
- 指揮官完整介面
- METHANE／初報
- 救護車派遣
- 消防車數量派遣
- 台電、欣桃瓦斯、外縣市支援
- 現場資源與指揮紀錄
- 一次檢傷官 START逐題導引
- 二次檢傷官 START逐題導引
- 待後送傷患
- 可用救護車
- 醫院收治患者量能
- 目前後送編組
- 後送紀錄
- 戰情中心即時同步

建議重新建立新的GitHub專案：
MCI-Trainer-TW-v2

上傳時，GitHub根目錄應直接看到：
package.json
server.js
render.yaml
public資料夾

Render部署：
1. Render選擇New → Blueprint
2. 連結新的MCI-Trainer-TW-v2 GitHub專案
3. Render讀取render.yaml後開始部署
4. 部署完成後使用新的公開網址測試

注意：
目前演練資料暫存在伺服器記憶體中，Render重新啟動後房間會消失。

v2.2.1 角色功能修正版：

v2.2.4 操作流暢版：

- 成功操作改為右下角短暫提示，不中斷流程。

v2.2.5 START簡化版：

- 題目與按鈕改成短句，適合手機快速操作。
- 保留傷勢小人圖與生命徵象輔助資訊。


v2.2.6 患者人數選項版：

- 可選6、12、24、36、48人。

v2.2.7：新增結果確認、取消檢傷、重新檢傷與教官重設功能，避免誤觸被算完成。

v2.3.0 檢傷級數顯示版：

v2.4.2 患者切換解鎖修正版：

v2.6.0 處置引導檢傷版：
- 成人依 START 流程。
- 兒童自動依 JumpSTART 流程。
- 大量出血患者必須先完成加壓止血。
- 無呼吸患者必須先打開呼吸道，再重新評估呼吸。
- JumpSTART 無呼吸且有脈搏時，必須給予5次人工呼吸。
- 記錄處置時間、操作者及處置內容。
- 檢傷結果仍須確認送出後才算完成。


v3.0.0 專業檢傷引擎：
- 成人依 START：行走、呼吸、打開呼吸道最多2次、RR>30、循環、意識。
- 兒童依 JumpSTART：行走、呼吸、打開呼吸道、脈搏、5次人工呼吸、RR14～45、AVPU。
- 大量出血先加壓止血。
- 孕婦與救災救護人員依規則列為第一優先紅色。
- 最後仍須確認送出才算完成。
